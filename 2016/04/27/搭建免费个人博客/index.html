搭建免费个人博客

三步完成免费个人博客搭建， 这是一篇小白也能看懂的文章。

1. 创建Github域名空间

1.1 注册
首先你需要注册一个Github账号，已有的可以下翻去1.2 创建仓库，注意username，这会影响到你的域名，你的域名将会是 username.github.io ，所以认真的取个名字吧。


注册过程可能需要验证你的邮箱，其他就不在赘述。

1.2 创建仓库
然后需要创建一个仓库(repository) 来存储我们的网站，点击首页任意位置出现的 New repository按钮创建仓库。

Respository name 中的 username.github.io (PS: io和com通用) 的 username 一定与前面的Owner 一致，记住你的 username下面会用到。


第一步就已经完成了，下面是安装。

2. 安装

Hexo可以说是目前最流行的博客框架了，基于Nodejs，更多信息可以google，下面需要安装的工具包括 Git，Nodejs，Hexo。

2.1 安装Git
通过安装程序来完成安装的，附上详细教程通道。

2.2 安装Nodejs
同样的，还是通过安装程序，附上详细教程通道。

2.3 安装hexo
以上所有都安装完成之后再安装Hexo

    $ npm install hexo-cli -g

ok,这些必须工具已经安装完成，那么我们就可以生成博客，上传至我们的Github 仓库了。

3. 编写/发布

那么我们开始创建博客吧，然后更改一些自定义的配置，或者加上自己喜欢的主题，发布一篇文章，最后发布到Github网站上(username.github.io)。

3.1 创建博客
username替换成1.2 创建仓库，执行下面代码，会创建出一个名为 username.github.io 的文件夹。

    $ hexo init username.github.io

3.2 更改博客主题/基础配置
如果你觉得现在的主题不好看，那么我们就去安装一个新的主题，切换至生成的Hexo 目录，安装主题

    $ cd username.github.io
    $ git clone https://github.com/litten/hexo-theme-yilia themes/yilia

3.2.1 基础配置
打开文件_config.yml,修改对应的键值（详细参数设置）：

    title: XXX的博客 ##你博客的名字
    subtitle: XXXXXX ##副标题，站点左上角
    description: XXXXXX ##给搜索引擎看的，对站点的描述，可以自定义
    author: XXX  ##你的名字
    language: zh-Hans    ##中国人不解释
    theme: yilia ##主题
    deploy:
    type: git    ##使用Git 发布
    repo: https://github.com/username/username.github.io.git  ##刚创建的Github仓库

3.3 发布第一篇文章
道路已经铺完，开始码字吧，创建一篇博文，如下：

    $ hexo new "Hello world!"
或者在source/_posts目录下直接创建一个Hello-world.md格式的文件，然后随意发挥。 最后测试一下： 
    $ hexo s

3.4 发布到Github
安装hexo-deployer-git自动部署发布工具


    $ hexo install hexo-deployer-git –save
    $ hexo clean && hexo g && hexo d  ##每次部署的步骤，按这三步来进行
    —————————————
    附上一些常用命令：
    hexo new”postName” #新建文章
    hexo new page”pageName” #新建页面
    hexo generate #生成静态页面至public目录
    hexo server #开启预览访问端口（默认端口4000，’ctrl + c’关闭server）
    hexo deploy #将.deploy目录部署到GitHub
    hexo help # 查看帮助
    hexo version #查看Hexo的版本

恭喜你，你的博客已经完成了，在浏览器中输入 http://username.github.io 就能够访问了。
